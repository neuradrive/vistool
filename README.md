    [x] CV Draw  
    [] GL Draw  
    [] YAML Config  
    [] ROS 1 Demo  
    [] ROS 2 Demo

    [] Protobuf    

2. Functions  
    [x] CV Draw  
    [x] CV Image Dump
    []  CV Save to Video  

### Get Start  

1. build from source & test  

```
    git clone https://github.com/neuradrive/vistool.git viztool
    cd viztool/vistool
    mkdir build
    cmake ..
    make -j$(nproc)
    ./vistest   # test
```
